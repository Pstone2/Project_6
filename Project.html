Teaching a Genetic Algorithm to Mine a Diamond in Minecraft

The goal of the project is to create and train a genetic algorithm to find and mine a diamond in the popular game Minecraft.

MineRL Library

The project will rely heavily on the MineRL library. MineRL allows users to run a free modified version of Minecraft provided by Microsoft that has been designed specifically to work with the MineRL library.

Visualization

The previously discussed MineRL library allows users to run the simulations headlessly, without a visualization, to increase computational speed. In addition the simulation can run the modified version of Minecraft provided by Microsoft and show the agent preforming the desired actions.

Genetic Algorithms

A genetic algorithm is a form of machine learning where the algorithm is given a task to compleate and then scored on how well they are compleating the task. Generally, how the model is scored is considered the fitness of the model. After running the model many times, the most fit model is selected as the basis for the next generation of modles. This process is repeated until the model has learned to compleate the task to the disired level.

Importance

While Minecraft is a videogame, it is the complexity of the problem that is important. In short, Minecraft is a sandbox game where every world is randomly generated and the path to the objective, mining diamonds, envolves many steps that do not inherntly appear to make progress towards getting diamonds. The overall concepts of making a genetic algorithm that is able to efficently do a task given random changing environment can have many real world applications.
